# Equipo FRC 3527 — BALAM Código del robot 2026

## Descripción general
Este repositorio contiene el código del robot para el Equipo FRC 3527 - BALAM para la temporada 2026.

## Estructura del repositorio
- src/main/java — código del robot (comandos y subsistemas)
- src/main/deploy - informacion util como rutas de autonomo que se cargan a la RoboRio

## Notas de desarrollo
- Sigue la estructura basada en comandos: un subsistema por grupo de hardware, comandos para los comportamientos.
- Mantén los mapeos de hardware y las constantes de ajuste en un único archivo de constantes/configuración.
- Prueba unitariamente la lógica matemática/de control; usa la simulación cuando esté disponible antes de desplegar.
